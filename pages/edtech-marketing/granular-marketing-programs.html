How a program runs

Built on audience layering.

Audience Layering

Every program starts with a curated, enriched contact list built from Ideal Customer Profile criteria: institution type, role, geography, technology stack. Built using Clay for enrichment, Apollo for database access, and LinkedIn Sales Navigator — then segmented in HubSpot before any outreach begins.

ABM Outbound Workflows

Structured, multi-step LinkedIn engagement sequences targeting specific accounts: visit profile, react to a recent post, connect, message — paired with a contact list, a HubSpot scheduling page, and a supporting email nurture sequence.
